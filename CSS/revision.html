HTML INTRODUCTION

What is HTML ?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for Web pages.
  • With HTML you can create your own Website.
  • HTML is easy to learn - You will enjoy it
  • HTML elements tell the browser how to display the content

A Simple HTML Document

<!DOCTYPE html>

<html>

<head>

<title> Page Title </title>

</head>

<body>

<h1>My First Heading</h1>

<p>My first Paragraph </p>

</body>

</html>

<!DOCTYPE html> defines that this document is an HTML5.

<html> element is the root element of an HTML page

<head> contains meta information about the HTML page

<title> element specifies a title for the HTML page

<body> element defines the document's body

<h1> element defines a large heading

<p> element defines a paragraph

Basic Tags

Header Tag

<h1> element defines a large heading

<h1>Heading 1 </h1>

<h2>Heading 2 </h2>

<h3>Heading 3 </h3>

<h4>Heading 4 </h4>

<h5>Heading 5 </h5>

<h6>Heading 6< /h6>

Paragraph Tag

<p> element defines a paragraph

<p>My first Paragraph </p>

<br> element defines a line break

<hr> element defines a thematic break

<p>My Second Paragraph </p>

Image Tag

The HTML <img> tag is used to embed an image in a web page.

The HTML <img> tag has two required attributes:

  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image

<img src = "image.png" alt = "HTML image" />

Anchor Tag

The HTML <a> tag defines a hyperlink.

The target attribute specifies where to open the linked document.

<a href = "URL" target = "_blank" >Google</a>

Video Tag

The HTML <video>lement is used to show a video on a web page.

<video width="320" controls >

<source src="movie.mp4" type="video/mp4">

</video>

<strong> - Important text

<em> - Emphasis text

<sup> - Superscript Text

<sub> - Subscript Text

HTML Lists

HTML lists allow web developers to group a set of related items in lists.

Unordered Lists

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

<ul>

<li>Coffee <li>

<li>Tea <li>

<li>Milk <li>

</ul>

Ordered Lists

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

<ol>

<li>Coffee <li>

<li>Tea <li>

<li>Milk <li>

</ol>

Description List

The <dl> defines the description list. the <dt>tag defines the term, and <dd>tag describes each term:

<dl>

<dt>Coffee <dt>

<dd>- black hot drink <li>

</ol>

HTML Tables

HTML tables allow web developers to arrange data into rows and columns.

Table Layout

<table>

<tr>

<th> Company </th>

<th> Contact </th>

<th> Country </th>

</tr>

<tr>

<td> Alfreds Futterkiste </td>

<td> Maria Anders </td>

<td> Germany </td>

</tr>

</table>

Table Tags

<table> - Defines a table

<th> - Defines a header cell in a table

<tr> - Defines a row in a table

<td> - Defines a cell in a table

<thead> - Groups the header content in a table

<tbody> - Groups the body content in a table

<tfoot> - Groups the footer content in a table

HTML Forms

Form Elements

<form action = "">

<label for = "id" > Username:</label>

<input type = "text" name = "name" id = "id" value = "John"/>

<label for = "id" > Password:</label>

<input type = "password" id = "id" value = "pass"/>

<button type = "submit"> Submit </button>

</form>

Form Attributes

action - defines the action to be performed when the form is submitted.

target - specifies where to display the response.

method - specifies the HTTP method to be used.

autocomplete - specifies whether a form should have autocomplete.

novalidate - specifies that the form-data should not be validated

Select Tag

The <select> element defines a drop-down list:

<select id = "id">

<option value = "volvo" >Volvo</option>

<option value = "Ferari" >Ferari</option

<option value = "audi" >Audi</option

</select>

Input Tag

<input type = "text"> - single-line text input field.

<input type = "password"> - defines a password field

<input type = "email"> - input fields that should contain an e-mail address.

<input type = "date"> - for input fields that should contain a date.

<input type = "radio"> - defines a radio button.

<input type = "checkbox"> - defines a checkbox.

<input type = "file"> - file-select field and a "Browse" button for file uploads.

<input type = "color"> - input fields that should contain a color.

Text Area

The <textarea> element defines a multi-line input field (a text area):

<textarea name="message" rows="10" cols="30" > The cat was playing in the garden. </textarea>
